# Rock-Paper-Scissors

## Description

An implementation of <a href="https://en.wikipedia.org/wiki/Rock_paper_scissors">"Rock, Paper, Scissors"</a>.

This will be played against the computer with the best of five winning.

The technologies used for this project are HTML, CSS and Javascript.

This is currently a minimalistic implementation. The focus is on developing:

* javascript skills
* the ability to implement solutions in a systematic manner

CSS styling and better HTML functionality are future features to be added.
